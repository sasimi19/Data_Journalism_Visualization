2016 한국언론진흥재단 ‘데이터시각화 실습’ 강의개요 입니다.  
강의슬라이드, 참조URL, 샘플코드, 샘플데이터 등을 포함하고 있습니다. 
궁금한 것들이 있으면 아래 연락처로 메일 혹은 쪽지 주세요
##Sample_Code

R package ‘ggmap’을 활용하여 일반 주소값을 위경도(lat/lon) 변환 코드와 실습한 결과물 Embed Code를 삽입할 수 있는 HTML 탬플릿을 포함하고 있습니다
